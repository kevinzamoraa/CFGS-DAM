# 🚀 CFGS en Desarrollo de Aplicaciones Multiplataforma (DAM)

Bienvenido al repositorio centralizado de mi trayectoria académica, técnica y profesional durante el estudio del **Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Multiplataforma (DAM)** en el CPIFP Bajo Aragón. 

Este espacio no solo recopila el código fuente, las tareas entregables y el material didáctico consolidado a lo largo de tres cursos académicos, sino que representa un testimonio real de superación, constancia y adaptabilidad, habiendo compaginado esta formación avanzada en _ingeniería_ de software con mi actividad laboral en la ONCE, desde noviembre de 2024, y un proceso de rehabilitación cognitiva, desde mediados del 2011.

---

## 📂 Estructura Organizativa del Repositorio

El repositorio está estructurado metodológicamente por módulos profesionales para facilitar la navegación técnica y la trazabilidad de las competencias adquiridas:

*   **`[Módulos de Programación y Entornos]`**: Proyectos e hitos desarrollados en **Java**, **Kotlin**, **_Lenguajes de Marcado (Xml y Html principalmente)_**, **CSS** y **_C Sharp_ (C#)**, aplicando principios de Programación Orientada a Objetos (POO), colecciones, concurrencia y optimización de algoritmos.
*   **`[Bases de Datos]`**: Diseños conceptuales, lógicos y físicos de bases de datos relacionales y no relacionales, incluyendo _scripts_ avanzados de SQL y configuraciones NoSQL orientadas a entornos en la nube.
*   **`[Desarrollo de Interfaces y Multimedia]`**: Diseños adaptativos, prototipados UX/UI y maquetación de vistas modernas utilizando Material Design 3, XML, Swing (_biblioteca gráfica en Java_) y lógica reactiva.
*   **`[Sistemas de Gestión Empresarial / Acceso a Datos]`**: Implementación de arquitecturas de software avanzadas como **MVVM** (Model-View-ViewModel) y **Clean Architecture**, asegurando una separación estricta de responsabilidades (Presentación, Dominio y Datos).
*   **[`[Proyecto Final: Temporis ⏳]`](https://github.com/kevinzamoraa/Temporis-Final_Project-DAM)**: Acceso directo al núcleo de mi proyecto de fin de ciclo. Una aplicación Android asíncrona enfocada en la gestión de la productividad con un estándar estricto de accesibilidad universal (Cumplimiento WCAG AAA, integración con TalkBack, escalado dinámico de fuentes y seguridad biométrica nativa).

## 📈 Filosofía de Desarrollo y Metodología

Cada proyecto alojado en este repositorio sigue una línea rigurosa de ingeniería de software:
1.  **Accesibilidad por Diseño:** Romper las barreras digitales creando software inclusivo adaptado a la diversidad funcional y cognitiva.
2.  **Calidad Automatizada:** Implementación de flujos de integración continua para mitigar la deuda técnica y asegurar el despliegue de código limpio.
3.  **Metodologías Ágiles:** Gestión de tareas mediante flujos de trabajo basados en tableros Kanban e hitos iterativos (Sprints).
